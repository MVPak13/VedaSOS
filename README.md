Наша задача разными схемами написать этот грёбаный бот, мы не входим в азарт, не добавляем лишних функций, не перегружаем бота, написали и тестим, добавим функции потом.

Бот должен добавляться в группы клиента и отправить сообщение о себе и о своём функционале.

После того как участник группы напишет команду /help начинается сбор неоходимой инфы для заполнения заявки Pyrus:

1.Имя группы (бот сам должен взять название из телеграмма)

2.Филиал (Бот должен отправить сообщение где просит написать филиал + кнопку нет филиала. Если участник написал название бот переходит на следующий вопрос. Если на кнопку, то он пишет в поле NULL и переходит к следующему)

3.Описание проблемы (Бот должен отправить сообщение где просит описать проблему, после того как учасник отправит сообщение бот отправит полное сообщение где описанны как заполненны поля и поле того заявка отправляется на Pyrus)
